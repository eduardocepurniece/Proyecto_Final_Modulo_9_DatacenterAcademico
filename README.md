# DataCenter Académico: Implementación y Administración de un Sistema de Base de Datos Empresarial

## Soluciones Empresariales RD
Este repositorio contiene el proyecto final colaborativo correspondiente al **Módulo 9**, el cual simula el funcionamiento de un departamento tecnológico dentro de una organización real. El objetivo es diseñar, implementar, administrar y explotar la infraestructura de datos centralizada para la empresa ficticia **"Soluciones Empresariales RD"**.

La plataforma permite gestionar de manera integral la información de clientes, empleados, productos, servicios, ventas e inventario, garantizando la integridad de los datos, la seguridad y la alta disponibilidad.

A lo largo del proyecto, los equipos deberán coordinarse para construir una solución empresarial completa que abarque desde el diseño de la base de datos hasta la implementación de mecanismos de seguridad, procedimientos de administración, documentación técnica y pruebas de funcionamiento.

## Estructura del Proyecto y Equipos

El desarrollo de este sistema se gestiona de forma colaborativa a través de roles especializados:

* **Equipo 1: DBA (Administradores del Servidor)**
    * *Enfoque:* Creación de la base de datos, usuarios, respaldos, scripts maestros y administración general.
* **Equipo 2: Diseño de Base de Datos**
    * *Enfoque:* Análisis del caso, modelo Entidad-Relación (ER), normalización, relaciones y documentación técnica inicial.
* **Equipo 3: Desarrollo SQL**
    * *Enfoque:* Creación de tablas, restricciones, consultas multitablas, subconsultas, vistas y procedimientos almacenados.
* **Equipo 4: Seguridad**
    * *Enfoque:* Roles, permisos, usuarios, vistas para control de acceso y políticas básicas de seguridad.
* **Equipo 5: QA (Control de Calidad)**
    * *Enfoque:* Pruebas, validación de resultados, revisión de scripts y detección de errores.
* **Equipo 6: Soporte y Documentación (Gestores del Repositorio)**
    * *Enfoque:* Administración de GitHub, mantenimiento del `README.md`, recopilación de evidencias, estructuración del manual técnico y preparación de la presentación final.
